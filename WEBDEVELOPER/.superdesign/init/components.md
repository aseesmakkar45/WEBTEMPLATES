# Shared UI Components

## btn-primary (CSS class)
Gradient orange-to-purple pill button. Used via className.

## btn-outline (CSS class)
Transparent pill button with subtle border.

## .service-card (CSS class)
Dark card with border, border-radius 16px, hover lift effect.

## .marquee-track (CSS class)
Infinite horizontal scroll animation container.

## .section-label (CSS class)
Small uppercase eyebrow label in orange.

## .glass (CSS class)
Glassmorphism surface with backdrop-filter blur.

## No component library — all custom inline styles + CSS classes.
